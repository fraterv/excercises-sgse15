Aufgabe 3 - Built-In-Direktiven

In dem hier verwendeten Controller wurde ein Array 'names' hinterlegt.
Es enthält einige Personendaten.

Aufgabenstellung:

Lassen Sie das Array mit Hilfe der Direktive 'ng-repeat' durchgehen.
Geben Sie hierbei die Variablen 'firstName' und 'lastName' aus dem Array aus.
Machen Sie außerdem jedes Element anklickbar.
Bei einem Klick auf das jeweilige Element, soll dieses mit Hilfe der fertigen Methode 'delete()' aus dem Array im Controller entfernt werden.

Hinweis: Übergeben Sie der Methode den Parameter '$index', der Funktionsaufruf lautet also 'delete($index)'.
'$index' enthält den Array-Index zu dem Element, welches angeklickt wird und somit gelöscht werden soll.
